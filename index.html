Raven Identity

What is Raven Identity ?
A library that gives you RavenDB storage provider for ASP.NET Identity.

It will integrate a RavenDB document database with ASP.NET Identity by handling all the underlying Identity API so that all the API remains unchanged but instead of a SQL Server database with EntityFramework a Raven Document database will be used.

What is RavenDB ?
RavenDB is the premier NoSQL database for .NET. It's a 2nd generation document database. Open source, speed-obsessed, and a joy to use. Schema-free. Just store your stuff.

What is ASP.NET Identity ?
The ASP.NET Identity system is designed to replace the previous ASP.NET Membership and Simple Membership systems. It includes profile support, OAuth integration, works with OWIN.

Chatter Club

What is Chatter Club ?
A proto-type built on the idea of simple no non-sense realtime & responsive chat application.

The realtime part of it is accomplished by using SignalR. The responsive part of it is acheived by using Semantic UI. Both of the frameworks are open-source.

What is SignalR ?
SignalR allows bi-directional communication between server and client. Servers can now push content to connected clients instantly as it becomes available. SignalR behaves as if the connection-less part of HTTP protocol has been turned into a connection-oriented one.

What is Semantic UI ?
User Interface is the language of Web. Semantic is a UI component framework based around useful principles from natural language.
